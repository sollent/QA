1. При валидном заполнении всех полей для поиска билетов, при нажатии на кнопку поиска,
   мы переходим к списку возможных билетов
Шаги:
   - заполним все поля валидными данными
   - нажимаем на кнопку "Найти"
Ожидаемый результат:
   - попадаем на страницу со списком возможным билетов


2. Заполнение полей "откуда" и/или "куда" несуществующими конечными пунктами, при поиске
   билетов
Шаги:
   - заполняем поля "откуда" и "куда" несуществующими странами
   - нажимаем на кнопку "Найти"
Ожидаемый результат:
   - подвсечивание полей, с предупреждением в ajax режиме
   - после обработки данных сервером, при переходе на страницу билетов видим предупреждение
     о невалидных данных или подсвечивание полей


3. При заполнение полей для поиска билета, при выборе даты вылета, нельзя выбрать дату возвращения с числами до даты вылета
Шаги:
   - при заполнение полей туда-обратно выбираем туда - 30 ноября, обратно - 28 ноября
   - нажимаем на кнопку "Найти"
Ожидаемый результат:
   - нет возможности выбрать дату возвращения, если она меньше даты вылета (блокировка соответствующих дат в календаре)


4. Попытка выбрать билет на прошедшую дату
Шаги: 
   - пытаемся выбрать прошлую дату отправения
Ожидаемый результат:
   - такие даты, заблокированны для клика


5. Пункт отправки и пункт назначения не должны совпадать
Шаги: 
   - заполняем поля "откуда" и "куда" одинаковыми странами
   - нажимаем на кнопку "Найти"
Ожидаемый результат:
   - подсвечивание полей в ajax режиме с просьбой ввести валидные данные
   - подсвечивание полей с просьбой ввести валидные данные после перезагрузки страницы


6. Попытка найти билет с датой на год вперед
Шаги:
   - при заполнении полей "откуда" и/или "куда" пытаемся выбрать дату, которая больше текущей на год
Ожидаемый результат:
   - в календаре нет возможности выбрать такую дату (календарь расчитан ровно на год, начиная с текущего месяца)


7. Проверка максимального кол-ва пассажиров на один билет
Шаги:
   - заполняя поле с пассажирами, по нажатию на кнопку "+", пытаемся выбрать максимально кол-во пассажиров
Ожидаемый результат:
   - мы не можем выбрать кол-во пассажиров больше установленного максимума (например 9)


8. Пытаемся найти билеты на несуществующие рейсы (например "Великий Новгород" - "Лондон")
Шаги:
   - в поля "откуда" и "куда" пишем "Нижний Новгород" и "Лондон"
   - нажимаем на кнопку "Найти"
Ожидаемый результат:
   - после перезагрузки страницы будет выдано сообщение а том, что был выбран не существующий рейс
   - после перезагрузки страницы будут подчеркнуты соответсвующие поля с пояснение об ошибке
   - после перезагрузки страницы невалидное поле будет заменено но возможную страну/город


9. Кол-во младенцев должно быть ровно кол-ву младенцев
Шаги:
   - при заполнии полей с пасажирами, пытаемся выбрать кол-во младенцев больше кол-ва взрослых
Ожидаемый результат:
   - кнопка "+" для младенцев становиться не кликабельна.


10. После поиска билетов, при нажатии на определенный билет, попадаем на станицу с подробной информацией о нём
Шаги:
   - заполняем все поля валидными данными
   - нажимаем на кнопку "Найти"
   - выбираем любой билет в списке найденных
Ожидаемый результат:
   - попадаем на страницу с более подробным описанием билета чем на странице со списком найденных. Нас не редиректит
     на сайт-партнер после нажатия на билет
